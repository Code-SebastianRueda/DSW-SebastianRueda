Etiquetas Basicas

Etiqueta Descripcion
head Es el encabezado de la pagina, se coloca antes de body
li Lista no ordenada, esta etiqueta va dentro de la etiqueta ul
header Representa un grupo de ayudas introductorias o de navegación
body Contiene y representa todo el contenido de la pagina web
html Esta etiqueta es una de las mas importantes contiene todo el codigo de la pagina
doctype Es la etiqueta que indica de que tipo es el documento
article Esta etiqueta contiene una composicion contenida en algun documento o pagina web
section Esta etiqueta divide una seccion de la pagina
footer Contiene el pie de pagina que se va a mostrar en la pagina web
img src Con esta etiqueta se pueden insertar imagenes
a href Con esta etiqueta se crean vinculos ya sea internos o externos a la pagina
ul Esta etiqueta representa que se va a crear una lista
nav Representa una sección de una página cuyo propósito es proporcionar enlaces de navegación, ya sea dentro del documento actual o a otros documentos
title Esta etiqueta contiene el titulo de la pagina web que aparecera en la respectiva pestaña
b Todo lo que se coloque dentro de estas etiquetas aparecera en negrilla
tr Esta etiqueta crea una nueva fila dentro de la tabla
td Esta etiqueta crea una nueva columna dentro de la tabla
br Salto de linea
strike Con esta etiqueta se tachara todo el texto que este detro de la etiqueta
i Con esta etiqueta se puede observar la letra inclinada
u Esta etiqueta se utiliza para subrayar el texto
h1 Esta etiqueta se utiliza para el titulo principal se puede colocar hasta el h6
p Esta etiqueta es utilizada para escribir un parrafo dentro de la pagina
table Con esta etiqueta se crean tablas dentro de la pagina
